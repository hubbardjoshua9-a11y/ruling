Module 2 – Choosing the right software for the job

There are dozens of programs on a typical work computer, but most everyday tasks only need a handful. The skill is knowing what each common application is *for*, so you can pick the right one quickly and open it the way your organisation expects.

In this module

  • 2.1 The common business applications and what each is for
  • 2.2 Matching the application to the task
  • 2.3 Opening the right application the organisation's way

Assessment Readiness – Module 2

Before moving on, you should be able to:

  • name the common business application types and say what each is used for
  • choose an appropriate application for a given task and explain your choice
  • open the right application the way your organisation does it, and open the correct file or start a new one.
