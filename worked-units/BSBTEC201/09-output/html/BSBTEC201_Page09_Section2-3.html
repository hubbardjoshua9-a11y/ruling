2.3 – Opening the right application the organisation's way

Organisations set up their software differently. Some have applications installed on the computer; others run them in a web browser as cloud apps. To open an application:

  • find it where your organisation keeps it: the desktop, the Start menu, the taskbar, or a saved bookmark in the browser
  • sign in with your organisational account if you are asked to
  • then open the correct existing file, or start a new one, as the task requires.

If you are not sure which system your organisation uses for a task (for example, whether leave is recorded in email or in an HR system), ask. Using the right system from the start keeps everyone's records in one place.

Examples across Australian workplaces
    • A hospitality venue administrator opens the bookings system from a browser bookmark and signs in with her venue account, rather than keeping bookings in a personal spreadsheet.
    • A construction company office uses installed Microsoft programs on the office computers, so the site administrator opens Word from the taskbar to start a site notice.
    • A government department officer opens the records management system from the desktop to file correspondence, because email is not where official records are kept.
