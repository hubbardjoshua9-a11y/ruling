Module 3 – Doing the work: enter, check, format and fix

This module is the core of the unit: actually working inside an application. The cycle is the same whatever the program: put the information in, check it, make it look right, and deal with any small problems that come up.

In this module

  • 3.1 Entering information and data accurately
  • 3.2 Checking and amending your work
  • 3.3 Formatting so it is fit for purpose
  • 3.4 When something goes wrong: help functions and simple faults

Assessment Readiness – Module 3

Before moving on, you should be able to:

  • enter information and data accurately, working from the source and following organisational conventions
  • check and amend your work using the application's tools and a read-back against the source
  • format a document, spreadsheet or presentation appropriately, using organisational templates where they exist
  • use an application's help functions, identify simple faults and fix them, and recognise when to escalate.
