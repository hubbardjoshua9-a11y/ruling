2.1 – The common business applications and what each is for

Most workplaces use a small set of application types. You do not need to master all of them; you need to know which one suits which job:

  • Word processor (for example Microsoft Word or Google Docs): letters, reports, memos, and any document that will be read or printed as text.
  • Spreadsheet (for example Microsoft Excel or Google Sheets): lists, numbers, calculations, budgets, and anything you need to sort or total.
  • Email and calendar (for example Microsoft Outlook or Gmail): sending messages, attaching files, and scheduling meetings.
  • Presentation (for example Microsoft PowerPoint or Google Slides): slides for a talk, a meeting, or an update shown on screen.
  • Database or records system (for example a customer management system, a patient records system, or a student records system): storing structured records so they can be searched and updated.
  • PDF reader and creator (for example Adobe Acrobat): fixed-format documents for sharing, printing or signing, where you do not want the layout to change.
  • Cloud file storage (for example SharePoint, OneDrive or Google Drive): storing and sharing files online so colleagues can reach them from anywhere.

These are the applications used for organising electronic information and data in most Australian workplaces. Some organisations use the Microsoft set, some use Google's, and many use a mix; the *types* are the same even when the brand differs.
