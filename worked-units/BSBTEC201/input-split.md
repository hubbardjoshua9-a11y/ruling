# BSBTEC201 — Resist/Integrate split
**Use business software applications** · researched 2026-06 (Warrant)
**Live anchors:** generative-AI document/spreadsheet drafting now native in mainstream office suites (Copilot in Microsoft 365, Gemini in Google Workspace, 2024–25); the routine production of business documents is AI-drafted by default. **Strongly AI-affected unit** — the competency has moved from keystroke production to directing and quality-checking AI output.

| Element / PC | Verdict | Why (research-anchored) | Modernisation note |
|---|---|---|---|
| 1.1 Identify task purpose, audience, format | **INTEGRATE** | Framing a document brief is now done *to* an AI tool; the skill is specifying intent and constraints | Specify the brief as an AI prompt; judge fitness of the result |
| 1.2 Select relevant technology & software | **INTEGRATE** | Tool selection now includes which AI assistant fits the task | Justify tool + AI-feature choice for the task |
| 1.3 Adjust workspace/furniture/equipment (ergonomics) | **RESIST** | Physical workspace setup; AI-thin, shown unaided | — |
| 2.1–2.2 Open application; enter information/data | **INTEGRATE** | Routine entry/drafting is AI-assisted in current practice | Use AI to draft; student directs and verifies |
| 2.3 Check & amend | **INTEGRATE** | The current competency is *quality-checking* AI output, not manual production | Annotate ≥2 problems in the AI draft and correct |
| 2.4 Format using application functions | **INTEGRATE** | Formatting is AI-generatable; the skill is judging whether it's right | Verify/fix AI formatting against the brief |
| 2.5 Use help functions for simple issues | **INTEGRATE** | "Help" is now an AI assistant; the skill is evaluating its answer | Use AI help, then judge the advice |
| 3.1 Review & edit; prepare for storage | **INTEGRATE** | Review is AI-augmented; judgment over the draft is the competency | Document corrections + reasoning |
| 3.2 Name & store document; exit | **RESIST** | Org file-naming/storage conventions; trivial, shown unaided | — |
