## The teaching point
The headline is what Ruling **didn't** do: it did not invent an "add AI" REVISE for a unit where AI doesn't touch the work. **AI-thin is signal, not a gap (RULE-33).** The only real fault it surfaced is an *authenticity / evidence-type* one (written responses can't evidence verbal/non-verbal performance, RULE-20) — caught by Axis A, with Axis B correctly silent. An operator that can't leave a current task alone is as broken as one that can't catch an obsolete one. CHCDIV001 is the guard against the new axis's false-positive direction.
